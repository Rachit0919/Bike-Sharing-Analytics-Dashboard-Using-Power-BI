# Bike Sharing Analytics – Power BI Project

## Overview
This project analyses bike-sharing data to understand yearly performance, revenue trends, and profit growth.

## Tools Used
- Power BI
- SQL (data source)
- DAX

## Key Insights
- Total Revenue and Profit overview
- Yearly performance comparison
- Year-over-year revenue growth (2021–2022)
